# sg studio — Implementation Plan

> **For agentic workers:** implement task-by-task; each ends with a green test or a stated manual check. Detail lives in the spec `docs/superpowers/specs/2026-06-22-sg-studio-design.md`.

**Goal:** A local browser studio to debug/run/prompt/fork/inspect-history and create/switch concept corpora against a live engine.

**Architecture:** `browser (React+htm+cytoscape via esm.sh, no build) ⇄ ws ⇄ server.js → studio.js → session.js → Graph`. The engine-facing `session.js`/`studio.js` carry the logic and are unit-tested; web layer is thin.

**Tech Stack:** Node CJS (lib/studio/*), `ws` (only added dep), `node:test`; frontend = React 18 + htm + cytoscape via import-map.

## Global Constraints
- Node 18+, **no build step** (lib/studio/* is CommonJS; frontend is CDN ESM, no bundler).
- Engine reached via `require('../index.js')` (the facade). **No engine-core changes.**
- The 111 existing tests must stay green.
- localhost only; single user.

## Tasks (backend = TDD; frontend = manual-verify against the realized protocol)
- [ ] **T1 protocol.js** — `OPS`/`EVENTS` name sets + a tiny test they're complete/frozen.
- [ ] **T2 session.js (core)** — `loadCorpus`/`state`/`mutate`/`run`; wires `onConceptApply`→`conceptApply` event, `onStabilize`→`stabilize`+`state`. Test: load common + Paris/Singapore seed → Distance casts, conceptApply emitted.
- [ ] **T3 session.js (history+concepts)** — `revisions`/`snapshot`/`rollback`/`diff`; `conceptTree`/`getConcept`/`validateConcept`/`patchConcept`/`addConcept`. Test: rollback restores; patchConcept re-evals; validateConcept flags a prose-edge.
- [ ] **T4 studio.js** — session registry (root + forks by id), `listCorpora` (scan `--root`), `fork`/`merge` (child sessions), the `ask` backend wiring. Test: fork → child stabilizes → merge reintegrates.
- [ ] **T5 server.js + protocol round-trip** — `http` static serve + `ws` bridge (op→studio, events→client). Test: connect over ws, `loadCorpus`+`mutate`, assert `conceptApply`/`state` events.
- [ ] **T6 `sg studio` CLI** — `lib/sg/cli.js` `studio` command: start server `--root --port --open`. Manual: `node bin/sg studio` serves the page.
- [ ] **T7 frontend infra** — `public/index.html` (import-map), `public/ws.js` (client: request/response by id + event sub), `public/app.js` (shell + Toolbar), `public/styles.css`. Manual: page loads, connects (● live), corpus switch works.
- [ ] **T8 GraphCanvas + Inspector** — cytoscape render from `state` (nodes/segments/facts), pulse on `conceptApply`; Inspector shows facts + cast-by + why. Manual: seed → see Distance edge, click → inspector.
- [ ] **T9 Timeline + TracePanel** — revisions slider, rollback, diff highlight; trace list (step through applies). Manual: rollback re-renders; trace lists applies.
- [ ] **T10 ConceptTree + ConceptEditor** — tree from `conceptTree`; JSON editor → `validateConcept` → `patch/addConcept` live. Manual: edit a concept assert → watch cascade.
- [ ] **T11 ForkTree** — fork/merge viz + session switch. Manual: fork → child appears → select → canvas switches.
- [ ] **T12 PromptConsole** — `prompt(text)` → answer-loop, stream tree + synthesized answer (needs `LLM_BASE`). Manual: prompt decomposes + synthesizes.

Frontend components live in `lib/studio/public/components/*.js`. Tests: `tests/integration/studio-session.test.js`, `tests/integration/studio-server.test.js`.
